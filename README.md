# Breast Cancer Detection using ML & Deep Learning

## 📌 Overview
This project compares multiple machine learning models and a deep learning (MLP) neural network for detecting breast cancer as malignant or benign.

## 🚀 Features
- Multiple ML models (Logistic Regression, Random Forest, SVM, etc.)
- Deep Learning using MLP Neural Network  
- Model comparison and performance evaluation  
- Visualization of results and confusion matrix  

## 🛠️ Tech Stack
- Python  
- scikit-learn  
- Pandas, NumPy  
- Matplotlib, Seaborn  

## 📊 Dataset
- Wisconsin Breast Cancer Dataset (Kaggle)  
- 569 samples, 30 features  

## ⚙️ Methodology
- Data preprocessing (normalization, encoding)  
- Train-test split (80/20)  
- Model training for multiple algorithms  
- Performance comparison  

## 📈 Results
- MLP Neural Network achieved highest accuracy (~98.2%)  
- Classical ML models also achieved >90% accuracy  

## ▶️ How to Run
1. Install dependencies  
2. Run Jupyter Notebook / Python script  
3. View model outputs and evaluation
